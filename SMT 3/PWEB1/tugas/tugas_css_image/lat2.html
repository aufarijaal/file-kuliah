Sejak didirikan pada 1975 di Amerika Serikat, Microsoft memiliki banyak unit bisnis. Tak hanya sistem operasi dan perangkat lunak(software),
tetapi ada pula unit bisnis mesin pencari (Bing), Konsol game (Xbox), layanan digital (MSN), e-mail (Outlook), sistem operasi mobile Windows phone, bahkan Microsoft membuat perangkat keras dan perangkat input.
